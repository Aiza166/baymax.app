# Baymax.app Project Report

**Baymax.app is a multi-agent AI career coach for Pakistan’s early tech talent.** It helps students and fresh graduates move through five practical career stages: Resume Builder, ATS Analyzer, Interview Coach, Job Scout, and Roadmap Planner.

## Demo Positioning

This deployed version is a challenge demo built to show how an accessible AI career coach can support early tech talent in Pakistan. It avoids paid-plan claims and focuses on the core evaluation flow.

## Five-Agent Pipeline

| Stage | Purpose |
|---|---|
| Resume Builder | Create or prepare a student-friendly resume. |
| ATS Analyzer | Check role fit, keywords, and improvement opportunities. |
| Interview Coach | Practise role-specific interview questions with feedback. |
| Job Scout | Review Pakistan-focused job matches and market fit. |
| Roadmap Planner | Convert gaps into a CSP-inspired 90-day learning roadmap. |

## Technical Engine

The roadmap planner uses the CSP/AI planning concept as the technical engine behind sequencing learning tasks, skills gaps, deadlines, and resources into a practical plan.
